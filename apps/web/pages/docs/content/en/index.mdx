### Code Example

```typescript
interface User {
  id: number;
  name: string;
  email?: string; // Optional property
}

// Function with type annotations
function greetUser(user: User): string {
  return `Hello, ${user.name}!`;
}

// Example usage
const newUser: User = {
  id: 1,
  name: "John Doe"
};

console.log(greetUser(newUser)); // Output: Hello, John Doe!
```

# Code Highlighting Example

```javascript {1,3} /console.log/
const x = 1;
const y = 2;
console.log(x + y);
```

In this example:
- First and third lines will be highlighted (via `{1,3}`)
- The word `console.log` will be additionally emphasized (via `/console.log/`)
- "javascript" will be shown in the corner
- github-dark theme will be applied
- Lines will be separated by a grid

You can customize styles for the `highlighted` and `word--highlighted` classes in your CSS to determine exactly how highlighted elements will appear.
